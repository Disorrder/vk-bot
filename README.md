# vk-bot
Node.js VK chat bot

## Requirements
 - Node.js >= 7.6.0
 - MongoDB >= 3.4
 - Redis

## Installation
 - `npm i`

## Running
Demo: `node index.js`
Developement: `nodemon index.js`
Production: `pm2 index.js`  - Not implemented yet!

### Running on Node.js 6.x
Demo: `npm run babel`
Developement: `npm run nodemon`
Production: `npm run pm2` - Not implemented yet!
